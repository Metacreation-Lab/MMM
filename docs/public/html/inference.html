Using the MMM model¶

This page presents how to use the MMM model to generate music.

Inference configuration¶

class mmm.InferenceConfig(bars_to_generate: dict[int, list[tuple[int, int, list[str]]]] | None = None, new_tracks: list[tuple[int, list[str]]] | None = None, autoregressive: bool = False, infilling: bool = False)¶

Configuration for the inference of MMM.

Specifies which bars will be infilled and which tracks will be generated. It also specifies the list of attribute controls to control the generation. All tracks will be used to condition the generation.

Parameters:

  • bars_to_generate – dictionary of couples [track_idx, [bar_start, bar_end, list of Attribute Controls]], where bar_start and bar_end are the extremes of the region to infill.

  • new_tracks – list of tuple containing the programs and attribute controls for the new tracks

The main inference entry point¶

mmm.generate(model: object, tokenizer: MMM, inference_config: InferenceConfig, score_or_path: Score | Path | str, generate_kwargs: Mapping | None = None) → Score¶

Use the model to generate new music content.

The method allows to infill specific bars or generate new tracks.

Parameters:

  • model – model used for generation

  • tokenizer – MMM tokenizer

  • inference_config – InferenceConfig

  • score_or_path – symusic.Score or path of the music file to infill.

  • generate_kwargs – keyword arguments to provide to the model.generate method. For Hugging Face models for example, you can provide a GenerationConfig using this argument.

Returns:

the infilled symusic.Score object.

StopLogitsProcessor¶

class mmm.StopLogitsProcessor(bar_start_token_id: int, eos_token_id: int, tokenizer: MusicTokenizer)¶

Custom transformers.LogitsProcessor implementation.

Allows stopping generation when enough content to infill bars is generated.

Parameters:

  • bar_start_token_id – ID of the token indicating the start of a bar.

  • n_bars_to_infill – number of bars to be infilled in this generation step.

  • eos_token_id – ID of the EOS (end of sequence) token. If the number

of bars reaches max_bars, the EOS token will be forced to stop generation.
